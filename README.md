# 方块深层撤离 / BLOCK EXTRACTION

一款依据《方块深层撤离》PRD 制作的零依赖 3D 网页可玩版。采用第一人称光线投射渲染，可直接在现代桌面浏览器运行。

## 启动

双击 `index.html` 即可运行。为获得更稳定的浏览器行为，也可以在本目录启动任意静态文件服务器。

## 操作

- `WASD` 移动，`Shift` 疾跑，鼠标观察和射击
- `E` 搜索、拾取核心、启动撤离
- `Q` 使用当前工具挖掘/建造，`F` 切换矿镐与泡沫建造器
- `1 / 2` 切换步枪和霰弹枪，`R` 换弹，`Esc` 暂停

## 完整游戏循环

搜索物资 → 与 PvE 威胁交战 → 挖墙改变路线 → 深入取得能源核心 → 返回西南升降机 → 坚守并撤离 → 结算信用点。局外记录保存在浏览器本地。

## 发布说明

当前交付是完整单机垂直切片，可部署到任意静态网站。PRD 中的 6–12 人 PvPvE、服务器权威结算、账号、匹配、反作弊及运营后台需要独立后端服务，不能由纯静态网页安全实现，属于正式联网版的下一研发阶段。
